# Notes App

ReactJS app to learn to perform different actions on notes.

## Sample Note:
```js
{
    id: 1,
    title: 'Note Title',
    description: 'Note Description'
    date: '12 Jan, 2020',
    category: 1
}
```
## Sample Category
```js
{
    id: 1,
    title: 'Programming',
}
```

## Filtering & Sorting
- Sorting via Date and Title
- Filtering via Category, Date
